2008年京都市立芸術大学大学院美術学科油画専攻修了
2019年まで高等専修学校で美術講師として勤務をしながら絵画制作を続ける。
この頃転職を考え、職業訓練校でWeb制作を学ぶことを決める。
2021年創造者デザイン専門学校Web専攻（2年間の職業訓練）終了
2021年よりWeb制作や画像処理などの仕事に就き経験を重ねる
未経験職種への転職は容易ではありませんでしたが、学びが仕事現場で役立つことを実感することがありました。職場では、解決できていない課題があった時、今自分にできる方法はないかを考え実践しました。今後も学びを続けながら、より良い解決や提案ができるように努めたいと考えています。
illustorator
実務では、文字やイラスト中心のメインビジュアル、バナー、ロゴデザイン、社内向けの機関誌制作。個人的には名刺やチラシのデザイン印刷の入稿が可能。パスを使ったイラストの制作可能。
photoshop
実務では、画像の補正や加工に使用。メインビジュアル、バナー、アイコンの制作。Webサイト用の素材はPhotoshopのみで済ませることが多い。
html
小規模なサイトからコーポレイトサイトまで制作可能。
css
小規模なサイトからコーポレイトサイトまで制作可能。sassやcss設計については書籍や動画で勉強している。
jQuery
スライダーやライトボックス、絞り込み検索など要望に応じて実装可能。
wordpress
自身のサイトや、友人のWebサイトの制作経験あり。前職では更新のみ行った。オリジナルテーマについては、練習で簡易なサイトを作ったことがある。
javascript
基礎的な学習はしている。実務の中では、ページトップ、文字の表示や非表示、アコーディオン、タブ、リンクメニューなど、必要に応じて調べながら実装した。今後はReactを習得したい。
